# Тестовые файлы

Эта директория содержит тестовые файлы для проверки системы антиплагиата.

## Описание файлов

- `file1.txt` - Первый тестовый файл с базовым текстом
- `file2.txt` - Второй тестовый файл с похожим текстом (частичное совпадение с file1.txt)
- `file3_original.txt` - Оригинальный текст об искусственном интеллекте
- `file4_plagiarized.txt` - Полностью скопированный текст из file3_original.txt (100% плагиат)
- `file5_unique.txt` - Уникальный текст без совпадений

## Использование

Эти файлы можно использовать для тестирования системы антиплагиата:
- Проверка обнаружения полного плагиата (file3 и file4)
- Проверка обнаружения частичного плагиата (file1 и file2)
- Проверка правильного определения уникального текста (file5)
